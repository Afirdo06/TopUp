TopUp Application

This is a PHP-based web application built with CodeIgniter 4 for managing game top-ups and related services.

Note: This application requires a PHP server to run. GitHub Pages only serves static files and cannot execute PHP code.

To run this application:

  • Clone the repository
  • Set up a PHP environment (XAMPP, WAMP, or similar)
  • Import the database from db_uts_10122491 .sql
  • Run the application using php spark serve
  • Access at http://localhost:8080
